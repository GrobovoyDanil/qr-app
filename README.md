# QR App

Приложение для генерации и сканирования QR-кодов с сохранением истории в
LocalStorage.

## Возможности
- Генерация QR-кодов из текста
- Сканирование QR-кодов через камеру
- История генераций и сканирований

## Стек
- React
- Vite
- React Router
- CSS Modules
- qrcode.react
- @yudiel/react-qr-scanner
